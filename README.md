# Online Restaurant System – Shruya Kitchen 🍽️

## Project Description

Shruya Kitchen is an online restaurant food ordering web application developed using Python and Flask. This system allows customers to browse food items, view the restaurant menu, and place orders through a simple and user-friendly interface.

---

## Features

* View restaurant menu
* Browse different food items
* Online food ordering
* Simple and user-friendly interface

---

## Technologies Used

* Python
* Flask
* HTML
* CSS
* JavaScript

---

## How to Run the Project

1. Install Python
2. Install Flask

pip install flask

3. Run the application

python app.py

4. Open a browser and go to

http://127.0.0.1:5000
